    The controller or widget who is rendering the view file.

    The data to be passed to the view.

  • $return Bool

Returns:

String:

The rendering result, or null if the rendering result is not needed.

Properties

cacheID

String

The identifier of the cache application component that is used to cache the compiled views. If null or empty, defaults to using a system.caching.CFileCache component.

Default: null

enableCaching

Bool

Value indicating whether to enable the caching of compiled views.

Default: true

engine

Mustache_Engine private

The underlying Mustache template engine.

enginePathAlias

String

The path alias of the directory containing the Mustache template engine.

Default: "ext.mustache.mustache.src.Mustache"

fileExtension

String

The extension name of the view files.

Default: ".mustache"

helpers

Mustache_HelperCollection

Values prepended to the context stack, so they will be available in any view loaded by this instance. Always null until the component is fully initialized.
